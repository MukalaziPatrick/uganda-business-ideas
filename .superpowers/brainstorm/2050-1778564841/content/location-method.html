How do users find businesses near them?

Location is the core feature — how should it work on a phone in Uganda where many people have basic internet?

A

Interactive Map (Google Maps style)

Show pins on a live map. User taps a pin to see the business. Needs good internet. Most visual and familiar.

  • Uses GPS to center on user
  • Pins by category (food, services, etc.)
  • Tap pin → business profile
  • Requires decent data connection
B

Area / District Dropdown

User picks their district or town from a list. Results show as cards. Works on slow internet, no GPS needed.

  • Select: Kampala → Nakawa → results
  • Fast on 2G/3G
  • No GPS permission needed
  • Less "discovery", more "search"
C

Both — Map + Area Filter

Map view for those with good internet. Area dropdown as fallback. User chooses what works for them.

  • Map tab + List tab on same page
  • GPS auto-detects district on map
  • List view works offline-ish
  • Best experience, more to build
